# 均线趋势策略 (MA200 Trend Strategy)

## 策略概述

这是最古老但也最经得起考验的方法。它的核心逻辑不是“多赚”，而是“少亏”。

## 核心理念

**长期趋势跟踪** - 顺势而为，在牛市持有，在熊市空仓。

## 📊 交易逻辑

### 买入/持有信号

- **收盘价 > 200 日均线 (MA200)**
- 满仓买入或继续持有

### 卖出/空仓信号

- **收盘价 < 200 日均线 (MA200)**
- 清仓（持有现金）

## 📈 策略优势

1. **躲避大熊市**

   - 完美躲过 2000 年互联网泡沫 (-80%) 和 2008 年金融危机 (-50%)。
   - 在大熊市中空仓，保护本金。

2. **复利效应**

   - 本金保住了，等牛市确立回来再进场，翻倍更快。
   - 长期回报率通常跑赢 Buy & Hold。

3. **简单有效**
   - 逻辑极其简单，无需复杂计算。

## ⚠️ 弱点与风险

1. **震荡市磨损**

   - 在震荡市 (如 2010-2016)，价格反复穿梭均线。
   - 会导致频繁“高买低卖”（左右挨耳光），跑输 Buy & Hold。

2. **滞后性**
   - 均线具有滞后性，会损失趋势初期的“鱼头”和末期的“鱼尾”。

## 📊 关键参数

| 参数     | 值     | 说明           |
| -------- | ------ | -------------- |
| 均线周期 | 200 天 | 长期趋势分界线 |

## 🎯 预期表现

- **市场环境**: 长期单边趋势（大牛市或大熊市）
- **表现**: 牛市跟涨，熊市空仓
- **胜率**: 较低（震荡市频繁止损），但盈亏比极高

---

**改进思路**: 为了防止频繁震荡，可以改为“月线收盘价 > 10 月均线”才操作，每月只交易一次。
