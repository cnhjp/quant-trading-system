# 波动率控制策略 (VIX Switch Strategy)

## 策略概述

利用 VIX（恐慌指数）作为红绿灯。SPY 和 VIX 有很强的负相关性。

## 核心理念

**恐慌时离场，平静时持有** - 美股的暴跌通常伴随着 VIX 的飙升。通过 VIX 过滤，可以躲过大部分“主跌浪”。

## 📊 交易逻辑

### 持有信号

- **VIX < VIX 的 50 日均线**
- 市场情绪相对稳定，持有 SPY/QQQ。

### 空仓信号

- **VIX > VIX 的 50 日均线**
- 市场恐慌情绪上升，空仓避险。

_(注：基础版逻辑为 VIX < 20 持有，VIX > 30 空仓。本策略采用进阶版均线逻辑，适应性更强)_

## 📈 策略优势

1. **避险神器**
   - 有效识别市场恐慌，躲过暴跌。
2. **自适应**
   - 使用 VIX 均线而非固定阈值，能适应不同波动率周期的市场环境。

## ⚠️ 风险提示

1. **VIX 飙升滞后**
   - 有时市场先跌 VIX 后涨，可能导致第一波下跌无法躲过。
2. **频繁切换**
   - 在 VIX 均线附近震荡时，可能频繁开平仓。

## 📊 关键参数

| 参数     | 值    | 说明           |
| -------- | ----- | -------------- |
| VIX 均线 | 50 天 | 波动率趋势判断 |

## 🎯 预期表现

- **市场环境**: 能够规避高波动下跌市
- **表现**: 在大跌时表现优异，但在牛市初期可能因为 VIX 未降下来而踏空一小段。

---

**注意**: 本策略依赖 VIX 数据，如果 VIX 数据缺失将无法生成信号（默认不操作）。
