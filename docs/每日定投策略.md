# 每日定投策略 (Daily DCA)

## 策略概述

Dollar Cost Averaging (DCA) 定投策略，将初始资金平分到回测周期的每一天。

## 核心理念

**时间分散成本** - 通过定期定额投资，平滑市场波动对成本的影响。

## 📊 交易逻辑

### 执行方式

- **每日固定投入** = 初始资金 / 回测总天数
- **买入价格** = 当日开盘价
- **持续买入** = 每天都买，从不卖出

### 对比基准

- **一次性投入** (Lump Sum) - 第一天全部买入并持有

## 📈 执行步骤

### 第 1 步：计算每日投入

```python
回测天数 = len(数据)
每日金额 = 初始资金 / 回测天数
```

### 第 2 步：逐日买入

```python
每日操作:
    投入金额 = 每日金额
    买入价格 = 今日开盘价
    买入股数 = 投入金额 / 买入价格
    累计股数 += 买入股数
    累计投入 += 投入金额
```

### 第 3 步：计算净值

```python
每日净值计算:
    已投入资金 = 累计投入
    剩余现金 = 初始资金 - 累计投入
    持仓市值 = 累计股数 × 收盘价
    总资产 = 持仓市值 + 剩余现金
```

### 第 4 步：基准对比

```python
基准净值 = (今日收盘价 / 首日开盘价) × 初始资金
```

## 💡 策略优势

1. **降低择时风险** - 不需要判断买入时机
2. **平滑成本** - 自动实现高买少、低买多
3. **纪律投资** - 机械化执行，避免情绪影响
4. **适合长期** - 时间越长，效果越明显

## ⚠️ 适用场景

- ✅ **长期投资** - 1 年以上效果更好
- ✅ **震荡上涨** - 定投优势最大
- ❌ **快速上涨** - 不如一次性投入
- ❌ **持续下跌** - 会不断买入下跌资产

## 📊 关键参数

| 参数     | 值       | 说明             |
| -------- | -------- | ---------------- |
| 初始资金 | 用户设定 | 总投资额         |
| 投资频率 | 每日     | 每个交易日都投入 |
| 投入金额 | 平均分配 | 总资金/天数      |
| 基准     | Lump Sum | 首日全仓买入     |

## 🎯 DCA vs 一次性投入

### DCA 的优势

- ✅ 降低单次大额投入的风险
- ✅ 不需要判断市场时机
- ✅ 心理负担小（分批进场）

### 一次性投入的优势

- ✅ 牛市中收益更高
- ✅ 充分利用复利效应
- ✅ 交易成本更低（只买一次）

## 📉 实际案例分析

### 场景 1：震荡上涨

```
市场走势: ↗↘↗↘↗
DCA表现: 优秀 ⭐⭐⭐⭐⭐
原因: 在低点买入更多，摊低成本
```

### 场景 2：快速上涨

```
市场走势: ↗↗↗↗↗
DCA表现: 一般 ⭐⭐⭐
原因: 越买越贵，不如首日全买
```

### 场景 3：持续下跌

```
市场走势: ↘↘↘↘↘
DCA表现: 较差 ⭐⭐
原因: 持续买入下跌资产，浮亏扩大
```

### 场景 4：V 型反转

```
市场走势: ↘↘↘↗↗
DCA表现: 优秀 ⭐⭐⭐⭐⭐
原因: 在底部买到更多股票
```

## 💰 资金管理

### 资金状态

```
总资产 = 持仓市值 + 现金余额

持仓市值 = 累计股数 × 当前价格
现金余额 = 初始资金 - 累计投入
```

### 逐步减少现金

```
第1天: 现金100%, 持仓 0%
第N天: 现金逐渐减少
最后一天: 现金0%, 持仓100%
```

## 📈 回测指标

| 指标     | 说明                             |
| -------- | -------------------------------- |
| 总收益率 | (最终净值 - 初始资金) / 初始资金 |
| 基准收益 | 一次性投入的收益率               |
| 总投入   | 累计投入的金额                   |
| 最大回撤 | 净值曲线的最大跌幅               |

## 🎯 预期表现

- **市场环境**: 任何环境都可用
- **交易频率**: 极高（每天交易）
- **胜率**: 不适用（只买不卖）
- **风险收益比**: 中等

## 💡 实战建议

1. **长期持有** - DCA 是长期策略，建议至少 1 年
2. **坚持不懈** - 即使市场下跌也要坚持投入
3. **对比基准** - 关注是否跑赢一次性投入
4. **情绪管理** - 下跌时更要加大投入（自动实现）

## ⚙️ 注意事项

1. **交易成本** - 本策略忽略了每日交易的成本
2. **实际操作** - 实盘中可能调整为周/月定投
3. **现金管理** - 剩余现金无收益（实际可能有利息）
4. **市场假设** - 假设每日都能按开盘价成交

---

**总结**：定投是"懒人投资法"，适合希望降低择时风险的长期投资者。虽然不一定能跑赢一次性投入，但能提供更好的心理舒适度。
