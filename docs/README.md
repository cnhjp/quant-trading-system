# 策略文档索引

本文件夹包含所有交易策略的详细说明文档。

## 📚 策略列表

### 1. [流动性掠夺策略](流动性掠夺策略.md)

- **英文名**: Liquidity Grab (SFP)
- **类型**: 反转策略
- **适用**: 震荡市、区间突破
- **核心**: 捕捉假突破后的反转机会

### 2. [趋势共振策略](趋势共振策略.md)

- **英文名**: Trend Confluence
- **类型**: 趋势跟随
- **适用**: 牛市趋势、低波动期
- **核心**: VWAP + VIX 双重过滤

### 3. [均值回归策略](均值回归策略.md)

- **英文名**: Mean Reversion (RSI)
- **类型**: 均值回归
- **适用**: 震荡上涨、趋势回调
- **核心**: RSI 超卖 + MA200 趋势过滤

### 4. [每日定投策略](每日定投策略.md)

- **英文名**: Daily DCA (Dollar Cost Averaging)
- **类型**: 被动投资
- **适用**: 长期投资、降低择时风险
- **核心**: 时间分散成本

### 5. [金字塔网格策略](金字塔网格策略.md)

- **英文名**: Pyramid Grid Strategy
- **类型**: 网格交易 + 金字塔加仓
- **适用**: 震荡市、深度回调后反弹
- **核心**: 分层加仓 + LIFO 止盈 + 永久底仓

### 6. [均线趋势策略](均线趋势策略.md)

- **英文名**: MA200 Trend Strategy
- **类型**: 趋势跟踪
- **适用**: 长期单边趋势
- **核心**: 200 日均线择时，牛市持有熊市空仓

### 7. [月底效应策略](月底效应策略.md)

- **英文名**: Turn of the Month Strategy
- **类型**: 日历效应 / 市场异象
- **适用**: 长期稳健投资
- **核心**: 月底月初持有，其余时间空仓

### 8. [波动率控制策略](波动率控制策略.md)

- **英文名**: VIX Switch Strategy
- **类型**: 风险控制
- **适用**: 规避暴跌
- **核心**: VIX 恐慌指数择时
